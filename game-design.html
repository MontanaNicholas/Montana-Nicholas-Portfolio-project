Fly Catcher

Code Software: Visual Studio/SFML

Jungle Explorer

This game was called "Jungle Explorer". The objective was to navigate through the 'jungle', avoiding enemies, to make it to the other side, which will lead the player to a 'level 2' ground.

Art Software: Aseprite
Code Software: Visual Studio/SFML

Lilypad Loot

This game was called "Lilypad Loot". The objective was to navigate across the river to reach the other side, while collecting coins and avoiding the rocks. Hitting a rock would cause the player to lose health (health bar displayed at the top of the screen) and collecting coins would add to the players 'score'. The joystick feature also made it playable on mobile devices.

Art Software: Aseprite
Code Software: Visual Studio/SFML
